EasyAsset Finance - home page (standalone, no repo)
====================================================

The page is fully self-contained (all CSS, JS and images are inline),
so there is nothing to build.

1) FASTEST - just view it
   Double-click index.html. It opens in your browser and works.

2) LOCAL SERVER (optional, gives you http://localhost:8000)
   Double-click start.cmd
   (needs internet the first time to fetch the tiny server)

3) PUT IT LIVE ONLINE - free, no GitHub repo
   Open https://app.netlify.com/drop and drag THIS whole folder
   onto the page. You get a public URL in ~10 seconds.

To update: replace index.html with your latest design and refresh.
